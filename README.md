# 🌾 AgroMedius

**AgroMedius** is a modern, AI-powered agricultural marketplace designed to bridge the gap between farmers and wholesalers. Built with a sleek, dark-themed interface, it provides real-time market data, AI-driven crop health analysis, and seamless direct messaging.

## ✨ Key Features

-   **📊 Live Market Tracking**: Real-time price updates for various crops across different markets with interactive trend visualizations (Bullish/Bearish).
-   **🤖 AI Crop Health Doctor**: Upload images of crops to get instant AI-powered health analysis and recommendations.
-   **🏫 Secure Marketplace**: Direct connection between farmers and wholesalers with role-based dashboards.
-   **💬 Real-time Messaging**: Built-in chat system for negotiation and coordination.
-   **📋 Order Management**: Track bookings, payment statuses, and delivery progress in one place.
-   **🔔 Smart Notifications**: Instant alerts for new orders, price changes, and messages.

## 🚀 Tech Stack

-   **Frontend**: React.js, Tailwind CSS, Lucide Icons, Vite
-   **Backend**: Node.js, Express
-   **Database**: SQLite (managed with Sequelize ORM)
-   **AI**: Mock ML integration for crop health and purchase suggestions

## 🛠️ Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vijaysag/AgroMedius.git
   cd AgroMedius
   ```

2. **Setup the Backend**:
   ```bash
   cd agro_server
   npm install
   # Create a .env file with JWT_SECRET and PORT
   node index.js
   ```

3. **Setup the Frontend**:
   ```bash
   cd ../agro_client
   npm install
   npm run dev
   ```
